Serves a shared worker plugin using explicit endpoint and authentication configuration.

This is primarily useful for tests and custom worker launchers. Relay-spawned workers should normally use [`serve_plugin`](/reference/api/rust-library-reference/nemo-relay-worker/fn-serve-plugin) or [`serve_plugin_arc`](/reference/api/rust-library-reference/nemo-relay-worker/fn-serve-plugin-arc).

## Errors

Returns an error when the endpoint configuration is invalid or the gRPC server fails.
