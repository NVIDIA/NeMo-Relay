Data-driven LLM model pricing used to layer cost estimates onto usage.

Model pricing is deliberately separate from response normalization so adding providers, aliases, or cache-accounting rules does not require editing [`AnnotatedLlmResponse`](/reference/api/rust-library-reference/nemo-relay/codec/response/struct-annotatedllmresponse).
