Provider-surface detection, best-effort normalization, and construction of the matching built-in codecs from a raw payload, surface, or codec name.

## Enums

- [ProviderSurface](/reference/api/rust-library-reference/nemo-relay/codec/resolve/enum-providersurface): A built-in provider request/response surface.

## Functions

- [detect_request_surface](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-detect-request-surface): Detect the request surface from a raw request body by top-level key.
- [detect_request_surface_with_hint](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-detect-request-surface-with-hint): Like [`detect_request_surface`](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-detect-request-surface), but a recognized `provider_hint` resolves the one ambiguous shape (an Anthropic request without a top-level `system`, otherwise read as OpenAI Chat). Today, only the exact hints `"anthropic"` and `"anthropic.messages"` change detection; `None` or any other value is ignored and detection stays shape-only.
- [detect_response_surface](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-detect-response-surface): Detect the response surface from a raw provider response, classifying only when exactly one built-in shape matches (the built-in codecs accept minimal objects, so decode success alone is not a reliable classifier).
- [normalize_request](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-normalize-request): Best-effort decode of a raw request into [`AnnotatedLlmRequest`](/reference/api/rust-library-reference/nemo-relay/codec/request/struct-annotatedllmrequest) (fail-open).
- [normalize_request_with_hint](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-normalize-request-with-hint): Like [`normalize_request`](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-normalize-request), but a recognized `provider_hint` can disambiguate provider request shapes that are otherwise identical.
- [normalize_response](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-normalize-response): Best-effort decode of a raw response into [`AnnotatedLlmResponse`](/reference/api/rust-library-reference/nemo-relay/codec/response/struct-annotatedllmresponse) (fail-open).
- [request_codec](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-request-codec): Constructs the built-in bidirectional request codec ([`LlmCodec`](/reference/api/rust-library-reference/nemo-relay/codec/traits/trait-llmcodec)) for a surface.
- [response_codec](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-response-codec): Constructs the built-in decode-only response codec ([`LlmResponseCodec`](/reference/api/rust-library-reference/nemo-relay/codec/traits/trait-llmresponsecodec)) for a surface.
- [streaming_codec](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-streaming-codec): Constructs a fresh, single-use streaming codec ([`StreamingCodec`](/reference/api/rust-library-reference/nemo-relay/codec/streaming/trait-streamingcodec)) for a surface.
- [supported_codec_names](/reference/api/rust-library-reference/nemo-relay/codec/resolve/fn-supported-codec-names): The canonical codec names of every built-in provider surface, for config validation and "supported codec" messages.
