Normalized LLM request data types. LLM request codec types and trait.

This module defines the [`AnnotatedLlmRequest`] type system for structured LLM request representation and the [`crate::codec::traits::LlmCodec`] trait for bidirectional translation between opaque [`crate::api::llm::LlmRequest`](/reference/api/rust-library-reference/nemo-relay-types/api/llm/struct-llmrequest) payloads and typed form.
