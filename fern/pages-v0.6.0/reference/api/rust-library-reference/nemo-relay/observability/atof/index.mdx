Agent Trajectory Observability Format (ATOF) JSONL exporter support for NeMo Flow.

The [`AtofExporter`](/reference/api/rust-library-reference/nemo-relay/observability/atof/struct-atofexporter) registers as an event subscriber and writes each canonical NeMo Relay Agent Trajectory Observability Format (ATOF) event as one JSON object per JSONL line.
