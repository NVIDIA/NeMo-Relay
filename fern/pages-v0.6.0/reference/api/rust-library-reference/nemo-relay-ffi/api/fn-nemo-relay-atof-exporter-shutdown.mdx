Outside a native subscriber callback, waits for queued subscriber delivery, then flushes the configured file sink or asks the configured stream sink to drain and close up to its timeout.

A re-entrant call does not establish the delivery barrier. A stream timeout is logged and does not by itself return an error.

## Safety

`exporter` must be a valid, non-null pointer.
